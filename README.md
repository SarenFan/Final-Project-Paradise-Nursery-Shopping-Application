# Paradise Nursery Shopping Application

## Project Description
Paradise Nursery is a React-based e-commerce web application for an online plant shop. It allows users to browse a variety of houseplants, view plant details (images, names, descriptions, and prices), add items to a shopping cart, and manage their cart dynamically. 

## Features
* Interactive navigation bar
* Dynamic product listing grouped by plant category
* Functional shopping cart that calculates total items and costs
* Ability to increase, decrease, or remove items from the cart

## Technologies Used
* React
* Redux (State Management)
* HTML/CSS
